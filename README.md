Pizza Shop Sales Analysis Using SQL

Project Overview

This project analyzes pizza shop sales data using SQL to generate business insights and solve business problems.

Dataset

The dataset contains information about pizza orders including order date, time, pizza category, quantity sold and price.

SQL Concepts Used

- JOIN
- GROUP BY
- ORDER BY
- LIMIT
- Subqueries
- Window Functions

Analysis Performed

1. Category-wise sales analysis
2. Pizza name-wise sales
3. Quantity-wise sales
4. Hourly sales analysis
5. Daily sales analysis
6. Top selling pizzas

Business Insights

- Identify the most popular pizza categories
- Find peak sales hours
- Determine top selling pizzas
- Analyze daily order trends

Tools Used

- MySQL
- SQL
